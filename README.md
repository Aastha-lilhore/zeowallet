# 💸 ZeoWallet

ZeoWallet is a modern student budget tracker built with Next.js, Firebase and Tailwind CSS.

## 🚀 Live Demo

https://zeowallet.vercel.app

## Features

* 🔐 Firebase Authentication
* 📊 Dashboard Overview
* 💰 Expense Tracking
* 🎯 Savings Goals Tracker
* 📅 Budget Planner
* 📈 Analytics Dashboard
* 👤 User Profile
* ⚙️ Settings Page
* 🌙 Modern Dark UI

## Tech Stack

* Next.js 15
* React
* TypeScript
* Tailwind CSS
* Firebase Authentication
* Firestore Database
* Recharts
* Vercel

## Run Locally

```bash
npm install

npm run dev
```
