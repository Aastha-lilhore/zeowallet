# 💰 ZeoWallet

ZeoWallet is a modern personal finance management application built for students to track expenses, manage budgets, set savings goals, and visualize spending habits.

## 🚀 Features

### 🔐 Authentication

* User Signup/Login
* Firebase Authentication
* Protected Routes

### 📊 Dashboard

* Monthly Budget Overview
* Total Expenses
* Savings Summary
* Recent Transactions

### 💸 Expense Tracker

* Add Expenses
* Delete Expenses
* Search Expenses
* Categorize Expenses

### 🎯 Savings Goals

* Create Savings Goals
* Track Progress
* Add Savings
* Delete Goals

### 📅 Budget Planner

* Monthly Income Planning
* Budget Calculation
* Remaining Balance Tracking

### 📈 Analytics

* Expense Analytics Charts
* Spending Visualization

### 👤 Profile

* User Information Display

### ⚙️ Settings

* Notifications Toggle
* Account Information
* Logout

## 🛠️ Tech Stack

* Next.js 15
* TypeScript
* Tailwind CSS
* Firebase Authentication
* Firebase Firestore
* Recharts

## 📱 Responsive Design

The application is optimized for:

* Desktop
* Tablet
* Mobile Devices
